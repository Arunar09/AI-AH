Core AWS Services
💻 Compute Services
Amazon EC2
Scalable virtual servers in the cloud
💡 Hosting web applications, running enterprise software
AWS Lambda
Serverless compute, run code without servers
💡 Event-driven functions, backend for web/mobile apps
Amazon ECS
Container orchestration service
💡 Deploying Docker applications at scale
Amazon EKS
Managed Kubernetes service
💡 Running Kubernetes workloads on AWS
💾 Storage Services
Amazon S3
Scalable object storage for the internet
💡 Data lakes, backup & restore, static website hosting
Amazon EBS
Block storage for EC2 instances
💡 Database storage, boot volumes for EC2
Amazon EFS
Scalable file storage for EC2
💡 Content management, media processing, shared file systems
Amazon S3 Glacier
Low-cost archival storage
💡 Long-term data archiving, regulatory compliance
🗃️ Database Services
Amazon RDS
Managed relational databases
💡 MySQL, PostgreSQL, SQL Server, Oracle, MariaDB
Amazon DynamoDB
Fast NoSQL database service
💡 Mobile backends, gaming, IoT, real-time applications
Amazon Aurora
High-performance relational database compatible with MySQL and PostgreSQL
💡 High-throughput applications, enterprise databases
Amazon Redshift
Cloud data warehouse service
💡 Business intelligence, analytics, data warehousing
🌐 Networking & Content Delivery
Amazon VPC
Isolate your resources in a virtual network
💡 Private cloud environments, secure network segmentation
Amazon Route 53
Scalable cloud Domain Name System (DNS)
💡 Domain registration, DNS routing, health checks
Amazon CloudFront
Fast content delivery network (CDN) service
💡 Global content distribution, website acceleration
Amazon API Gateway
Create, publish, and manage APIs at scale
💡 Serverless APIs, microservices integration, mobile app backends
⚙️ Management & Governance
AWS IAM
Manage access to AWS resources securely
💡 User management, role-based access control, security
Amazon CloudWatch
Monitor resources and applications
💡 Performance monitoring, logging, alarms, dashboards
AWS CloudFormation
Infrastructure as Code service
💡 Automating infrastructure provisioning, resource management
AWS Organizations
Centralized management of multiple AWS accounts
💡 Consolidated billing, policy enforcement, account governance
🛠️ Developer Tools
AWS CodePipeline
Continuous delivery service
💡 Automating release processes, CI/CD pipelines
AWS CodeBuild
Managed build service
💡 Compiling code, running tests, creating build artifacts
AWS CodeDeploy
Automate code deployments
💡 Deploying applications to EC2, Lambda, on-premises servers
AWS Cloud9
Cloud-based IDE
💡 Collaborative coding, serverless development
